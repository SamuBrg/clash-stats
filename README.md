# Card Stats for Clash Royale

This is a simple, lightweight web app to track your Clash Royale stats and analyze card performances in real-time.

I started this project to combine my personal interests with improving my Python skills. It's my first deep dive into working with live REST APIs, setting up a relational database (SQLite), and building a frontend. To keep things clean and efficient, I chose Streamlit over Flask for the dashboard interface. I'll continue to update this project and plan to fully host it in the future.

---

## Features
- **Winrate Analysis:** Calculate and visualize winrates for each card (both your own and your opponents').
- **Recent Performance:** Track your stats over your most recent battles.
- **Usage Rates:** Compare how often you play specific cards versus how often you face them.

## Upcoming
- **Deck Library:** A place to save your favorite decks. The in-game limit of 10 decks is never enough!
- **Game Mode Filters:** The ability to filter the data set for stat calculations (e.g., Trophy Road, Ranked/Path of Legend, Events).
